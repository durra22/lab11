Credit Card Customer Dataset

This dataset contains credit card customer information and their transaction behavior. It is commonly used for data analysis, customer segmentation, clustering, and machine learning projects.

The dataset includes 8,950 customer records with 18 features related to:

Account balance
Purchase behavior
Cash advance usage
Credit limits
Payment history
Purchase frequency
Installment purchases
Full payment percentage
Customer tenure

Example Features
BALANCE — Remaining account balance
PURCHASES — Total purchase amount
CASH_ADVANCE — Cash withdrawn in advance
CREDIT_LIMIT — Credit card limit
PAYMENTS — Total payments made
PURCHASES_FREQUENCY — Frequency of purchases
TENURE — Number of months as a customer

Possible Use Cases
Customer segmentation
Spending behavior analysis
Credit risk analysis
Clustering algorithms
Data visualization
Predictive modeling

The dataset is suitable for beginners and intermediate-level data science and machine learning projects.
